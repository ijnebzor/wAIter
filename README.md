# wAIter

**The wait is over.**

AI-run operations, peer-to-peer supplier marketplace, and self-paying invoices for restaurants who already know what they're doing.

## What this repo is

This is the public landing site and interactive demo for wAIter — a hospitality operations platform built around three ideas:

1. **AI Manager** — runs alongside your existing stack, surfaces what your tools hide.
2. **Sourcing Marketplace** — peer suppliers, smart-contract settlement, portable reputation.
3. **Self-paying invoices** — money moves itself. You never chase a payment again.
